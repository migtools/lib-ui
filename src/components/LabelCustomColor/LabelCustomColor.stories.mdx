# LabelCustomColor

A wrapper for PatternFly's <a href="https://www.patternfly.org/v4/components/label">Label</a> component that supports
arbitrary custom CSS colors (e.g. hexadecimal).

This component attempts to solve the color contrast issue by applying the given color as a border color and using
CSS magic to determine a lightened background color.
